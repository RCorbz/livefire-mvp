# Style Guide

This document outlines the style guide for our project. It ensures a consistent and cohesive look and feel across the entire application.

## Colors

- **Primary:** #3B82F6 (Blue)
- **Secondary:** #10B981 (Green)
- **Accent:** #F59E0B (Amber)
- **Neutral:** #6B7280 (Gray)

## Typography

- **Font Family:** "Inter", sans-serif
- **Headings:** "Poppins", sans-serif

## Spacing

- Use a consistent spacing scale based on multiples of 4.

## Corner Radius

- Use a consistent corner radius of 8px for all elements.
